# ANCIENT-HACK

V1RUS INTERNAL ESP
Продвинутый модуль визуализации, создающий полный контроль над окружением.
Поддерживает box-ESP, линии, ники, дистанцию, оружие и chams-эффекты.
Создан для максимальной информативности и чистой минималистичной подачи.

1. Запускаем инжектор smi_gui
2. Заходим в игру ANCIENT
3. Когда зашли в игру, инжекторе нажимаем Refresh, автоматом находит игру
4. В Assembly to inject вставляем длл чита ANCIENTXESP.dll
5. Class name пишем Loader
6. Method name пишем Init
7. Нажимаем на кнопку инжекта
